# 🌐 Shahanas P A — Portfolio Website

A personal portfolio website showcasing my projects, skills, and contact info, built with **HTML, CSS & JavaScript**.

## Features
- Responsive design with mobile hamburger menu
- Smooth scroll reveal animations
- Animated skill progress bars
- Rotating avatar frame with glow effect
- Contact form with submission feedback
- Dark gold theme with noise texture
